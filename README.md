# README
## Shelter API

### Development Setup
- Ruby version 2.7.1
- Rails version 6.0.4
### System Level Dependencies:
- PostgreSQL 14.1
### Project Level Dependencies
- bundle install
### Setting up the Database
```
rails db:create
```
```
rails db:migrate
```
```
rails db:seed
```
### Start the Rails Server
```
rails server
```
### Running Tests
```
bundle exec rspec
```
